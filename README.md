Notes Manager 🚀
A full-stack, aesthetically pleasing Notes Management System built with React, Node.js, Express, and SQLite.

🔗 Live Links
Frontend: https://stirring-trifle-72f76b.netlify.app/
Backend API: https://notes-manager-p28t.onrender.com/api
✨ Features
📝 Create, Read, Update, Delete (CRUD): Full control over your notes.
🎨 Premium UI/UX: Dark mode, glassmorphism, and smooth micro-animations.
🏷️ Categorization: Organize notes by categories (Personal, Work, Ideas, Study).
🔍 Search functionality: Quickly find notes by their title or content.
💾 Persistent Storage: All data is securely saved in a SQLite database.
🛠️ Technology Stack
Frontend: React.js, Vite, Vanilla CSS, Lucide React (Icons).
Backend: Node.js, Express.js.
Database: SQLite.
Deployment: Netlify (Frontend), Render (Backend).
🚀 How to Run Locally
Prerequisites
Node.js installed on your machine.
1. Clone the repository
bash

git clone https://github.com/fahmidhaabdulhakeem-dev/NOTES-MANAGER.git
cd NOTES-MANAGER
2. Start the Backend
bash

cd backend
npm install
npm run dev
(The backend server will start on http://localhost:5000)

3. Start the Frontend
Open a new terminal window in the root folder, then:

bash

cd frontend
npm install
npm run dev
(The frontend will start on http://localhost:5173)
